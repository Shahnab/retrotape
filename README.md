## ğŸ® How to Use

1. **Search for an Artist**: Type an artist name in the "Request Mix" notepad and click "CREATE MIX"
2. **Cassettes Appear**: 5 cassette tapes with songs will scatter on your desk
3. **Drag to Play**: Drag any cassette tape to the walkman to load it
4. **Control Playback**: Use the walkman buttons to play, pause, stop, or eject
5. **Customize**: Rotate tapes and the walkman, change backgrounds, adjust volume
